## 第一部分：基础一级指针（判断输出 + 画图）

### 题 1

```c
#include <stdio.h>
int main() {
    int a = 20;
    int *p = &a;
    *p = 66;
    printf("a=%d\n", a);
    printf("*p=%d\n", *p);

```

问题：输出结果是多少？画图说明内存变化。

a=20    //a=99

*p=66

说明：`*p=66` 解引用指针 p，直接修改 a 内存里的数据。


输出：99

## 第二部分：函数传一级指针（修改普通变量）

### 题 3

```c
#include <stdio.h>
void change(int *num) {
    *num = 1000;
}
int main() {
    int a = 10;
    change(&a);
    printf("%d", a);
    return 0;
}
```

问题：输出结果？如果函数参数改成 `int num`，调用 `change(a)`，结果会变成多少？为什么？

输出：1000

10

因为形参为指针时，修改的是该值的地址所在的内容，而当形参为一个普通变量时，传入的是一个值的副本，不会影响该值。

## 第三部分：二级指针核心题（你最疑惑的修改指针指向）

### 题 4（对标你原来的代码）

```c
#include <stdio.h>
void set(int **pp) {
    int temp = 888;
    *pp = &temp;
}
int main() {
    int a = 10;
    int *p = &a;
    set(&p);
    printf("%d", *p);
    return 0;
}
```

问题：

1. `set(&p)` 为什么不能写成 `set(p)`？
2. 函数里为什么写 `*pp = &temp`，不是 `**pp = &temp`？
3. 这段代码存在什么安全隐患？

说明：

pp 是`int**`二级指针，需要传入指针变量 p 的地址`&p`；p 是`int*`类型，类型不匹配；

`*pp`等价于 main 里的 p，给 p 赋值新地址；`**pp`等价于 a，是 int 数字，不能存地址；

temp 是函数局部变量，函数结束后销毁，p 变成悬垂野指针，输出随机垃圾值。

### 题 5

```c
#include <stdio.h>
void func(int **pp) {
    **pp = 500;
}
int main() {
    int a = 10;
    int *p = &a;
    func(&p);
    printf("a=%d", a);
    return 0;
}
```

问题：输出多少？`**pp = 500` 修改的是谁，和上一题 `*pp = 地址` 有什么本质区别？

修改的是a

说明：输出 a=500；

`**pp=500`：修改 p 指向的整型数据 a；

`*pp=地址`：修改指针 p 本身的指向，两个操作对象完全不同。
